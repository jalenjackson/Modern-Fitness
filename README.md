# Modern Fitness

### Development Environment
 
* Rails `5.1.4`
* rvm `1.29.3`
* Bundler `1.16.0.pre.2`

### Installation

Install the dependencies and start the server.
```sh
$ cd rgmfp-ror-website
$ bundle install
```

### Deployment
```sh
$ rails s
```

Verify the deployment by navigating to your server address in your preferred browser.

```sh
127.0.0.1:3000
```
